HTMLページ用

1.解析するページのHTMLタグ設定

<body> の直後に下記のコードを記述して下さい。全ページ共通です。
<body> の直後でなくても動作しますが、下に記述するほど精度が下がります。

<div id="tracker" style="position:absolute;visibility:hidden;">
<script type="text/javascript" src="$url_dir/tracker.js"></script>
<noscript><img src="$url_dir/write.php?guid=ON&act=img" width="0" height="0" alt="tracker"></noscript>
</div>

2.より正確に解析するHTMLタグ設定（任意）

上記のコードの代わりに下記のコードを記述して下さい。各ページのURLが必要です。
JavaScriptが実行出来ない環境でもアクセスページを取得可能になります。

<div id="tracker" style="position:absolute;visibility:hidden;">
<script type="text/javascript" src="$url_dir/tracker.js"></script>
<noscript><img src="$url_dir/write.php?guid=ON&act=img&url=そのページのURL" width="0" height="0" alt="tracker"></noscript>
</div>

PHPページ用

1.解析するページのPHPコード設定

PHPソースコードの全てのHTML出力の後に下記のコードを記述して下さい。
動作しない場合は全ての処理の前に記述して下さい。

<?php
include('$tracker_php');
?>

※上記で動作しない場合は下記に変更して記述して下さい

<?php
include('$write_php');
?>

2.より詳細に解析するHTMLタグ設定（任意）

上記のコードに加えて </body> の直前に下記のコードを記述して下さい。
「リンク先」「訪問回数」「解像度」「表示領域」が取得可能になります。

<script type="text/javascript" src="$url_dir/add.js"></script>

RSSフィード用

1.解析するRSSフィードのXMLタグ設定（RSS2.0）

</channel> の直前に下記のコードを記述して下さい。

<item>
<description>
&lt;img src="$url_dir/write.php?act=img&amp;url=RSSフィードのURL" width="0" height="0" alt="tracker" /&gt;
</description>
</item>

全ページ共通

1.テキストカウンターを表示するHTMLタグ設定（任意）

HTMLソースコードの表示させたい部分に下記のコードを記述して下さい。
「uu」「today」の部分は用途に応じて下記の文字列に変更可能です。

<script type="text/javascript" src="$url_dir/index.php/counter/uu/today/"></script>

ユニークユーザー数：uu　　ページビュー数：pv

累計カウント：total　　昨日カウント：yesterday　　今日カウント：today


2.パラメータ統計を表示するHTMLタグ設定（任意）

HTMLソースコードの表示させたい部分に下記のコードを記述して下さい。
「page」の部分は用途に応じて下記の文字列に変更可能です。
「年/月/表示件数/」を省略すると現在の年月のデータが表示されます。

<script type="text/javascript" src="$url_dir/index.php/clip/page/$y/$m/10/" charset="UTF-8"></script>

書式：/clip/パラメータ名/年/月/表示件数/

ページ：page　　リンク元：referrer　　接続元ドメイン：host_domain
検索語（単語）：search_word　　検索語（フレーズ）：search_words
ブラウザ：ua　 携帯：mobile　 OS：os　 解像度：display_size　 表示領域：client_size
